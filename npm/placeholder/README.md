# @barbacana/placeholder

This package exists solely to reserve the `@barbacana` npm scope and prevent typosquatting or impersonation. **It contains no functional code yet.**

Barbacana is an open-source WAF and API security gateway built on Caddy + Coraza + OWASP CRS v4. It is written in Go and is not distributed via npm.

- Project: <https://github.com/barbacana-waf/barbacana>
- License: Apache-2.0

A `@barbacana/cli` package is planned and will be published from a sibling directory under [`npm/`](../) (e.g. `npm/cli/`) in the main repository.
